Virtuelle Umgebung & Rekall installieren
  1. pacman -S virtualenv

    Bei Debian: apt install virtualenv-python

  2. virtualenv /pathToEnv

    Der Pfad ist frei wählbar!

  3. source /pathToEnv/bin/activate
  4. pip install rekall

    Es könnten Probleme beim Version von Wheel, pip oder Setuptools vorkommen.

    pip install --upgrade pip wheel setuptools

  5. rekall --help

    Sollte ein Fehler wegen der efilter Version aufkommen:

    pip install --upgrade rekall-efilter
